# 幾歲能退休？

假設每年存下的錢都投入大盤，來算算你的退休時間表。

**👉 [打開試算](https://brianhu-tw.github.io/fire/)**

## 功能

- 輸入年齡、資產、收入、支出，即時計算最早可退休年齡
- 悲觀 / 基準 / 樂觀三種報酬率情境模擬
- 互動式圖表：拖拉調整退休年齡與收入
- ETF 快速選單：0050、VOO、QQQ、VTI、VT，自動帶入歷史年化報酬率
- 所有金額以實質購買力（扣除通膨）呈現
- 輸入自動存入瀏覽器，重新整理不會消失
- 可產生分享連結，把你的試算結果傳給朋友

## 預設值

| 欄位 | 預設值 | 來源 |
|------|--------|------|
| 年收入 | NT$ 546,000 | 113 年台灣薪資中位數（主計總處） |
| 年支出 | NT$ 320,000 | 113 年台灣每人消費支出（主計總處家庭收支調查） |
| 年報酬率 | 7% | — |
| 通膨率 | 3% | — |

## 技術

- 單一 `index.html`，無建置步驟
- [Chart.js 4.x](https://www.chartjs.org/)（CDN 載入，含 SRI 驗證）
- CSS custom properties 深色主題
- localStorage 儲存使用者輸入

## 免責聲明

此工具僅供參考，不構成投資建議。實際報酬率會波動，當個方向參考就好。不含勞保、勞退、保險等，實際退休條件可能更寬裕。
