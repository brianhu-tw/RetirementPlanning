# CLAUDE.md

## Project Overview

A retirement planning simulator — a single-page Traditional Chinese (zh-TW) web app. Everything lives in one file: `index.html` (inline CSS + HTML + JS), no build step, no bundler, CDN-loaded Chart.js 4.x.

## Development

No build or test commands. Open `index.html` directly in a browser or use any local HTTP server.

## Rules

- All UI text is in Traditional Chinese (zh-TW). Maintain this convention.
- **Never push without explicit user approval.**
